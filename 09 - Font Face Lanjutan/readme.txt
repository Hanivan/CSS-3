Kepanjangan WOFF / WOFF2
Web Open File Format.

Dikembangkan oleh Mozilla yang dibuat khusus untuk website agar diload lebih cepat dibandingkan dengan format ttf. Walaupun woff sebetulnya adalah kompresi dari format ttf/otf

Cara merubah font yang berformat ttf/otf, bisa di konversi melalui website fontsquirell.com di bagian menu Generator
