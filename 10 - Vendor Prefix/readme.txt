Vendor Prefix adalah property CSS 3 Agar syntax yang kita buat didukung oleh semua browser

Penjelasan:
	Vendor Prefixes adalah sintaks khusus yang harus ditambahkan untuk beberapa property CSS

Sintaks spesifik untuk tiap browser/vendor itu berbeda

Macam-macam vendor prefixes:
 -webkit-<property-css3> => Untuk Chrome, Safari, Opera versi terbaru
 -moz-<property-css3> => Untuk Firefox
 -ms-<property-css3> => Untuk Internet Explorer
 -o-<property-css3> => Opera versi terdahulu

Kegunaan CSS Prefix adalah karena tidak semua browser support CSS 3. Pada saat materi ini dibuat

Untuk mengetahui property CSS3 apa saja yang butuh prefix. Kita bisa kunjungi shouldiprefix.com -- caniuse.com -- pleeease.io/play/ Atau bisa menggunakan plugin Emmet pada Code Editor
