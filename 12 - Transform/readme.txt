Transform adalah property yang memungkinkan kita dapat memanipulasi format visual dari element HTML. Property transform ini bisa 2D & 3D, tapi disini kita hanya mengimplementasikan pada 2D saja. Pada saat materi ini dibuat, mungkin property transform ini masih experimental
Penggunaan:
 transform: <fungsi-transform>;
Fungsi-fungsi transform:
 - scale => memperbesar/memperkecil
 - rotate => memutar
 - skew => membuat condong/miring
 - translate => mengubah posisi
